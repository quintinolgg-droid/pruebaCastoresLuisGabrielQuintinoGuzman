•	Lista de pasos para correr su aplicación.
Iniciar el Servidor MySQL: Asegúrese de que el servicio del servidor de MySQL esté activo.

Ejecutar Script de BD: Cargue y ejecute el script SQL (por ejemplo, schema.sql) para crear la base de datos y todas las tablas necesarias (movimientos, productos, usuarios, etc.).

Ejecutar la Aplicación (Opción NetBeans):

En NetBeans, haga clic derecho en el proyecto (web-mvc-castore).

Seleccione "Run" (Ejecutar).

Verificar Acceso: Abra un navegador web y navegue a la URL donde se despliega la aplicación (generalmente: http://localhost:8082).
